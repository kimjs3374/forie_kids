1. 예약 신청하기

월 단위 이용 신청을 접수하는 기본 메뉴입니다.

신청 페이지로 이동
  1. 신청 대상 월 선택 : 메인 화면에서 원하는 월 카드를 눌러 현재 접수 가능한 월을 선택합니다.
  2. 신청서 작성 : 이름, 동/호수, 연락처, 아이 인원수 등 필수 정보를 입력하고 개인정보 동의를 완료합니다.
  3. 신청 접수 : 신청이 완료되면 상태는 먼저 예약대기로 등록됩니다.
  4. 입금 확인 후 이용 : 관리사무소 입금 확인이 끝나면 해당 월은 날짜 지정 없이 자유롭게 이용할 수 있습니다.

2. 신청 조회

내 신청 상태를 확인하는 메뉴입니다.

신청조회로 이동
  1. 입력 정보 : 이름, 연락처, 동, 호수를 신청 당시와 동일하게 입력합니다.
  2. 확인 가능 항목 : 예약월, 현재 상태, 입금확인 여부, 입금확인일, 신청일시를 확인할 수 있습니다.
  3. 조회가 안 될 때 : 이름/연락처/동호수 중 하나라도 다르면 조회되지 않을 수 있으니 다시 확인해주세요.

3. 문의사항

입금 자동 반영이 늦거나 기타 문의가 있을 때 사용하는 메뉴입니다.

문의사항으로 이동
  1. 기본 정보 입력 : 이름, 연락처, 동/호수를 입력합니다.
  2. 요청 내용 작성 : 입금 날짜, 입금자명, 요청 내용을 자세히 적어주시면 관리자가 확인하기 쉽습니다.
  3. 답변 조회 : 같은 화면에서 답변조회 버튼을 누르면 내가 남긴 요청과 관리자 답변을 함께 볼 수 있습니다.
  4. 자동 반영 지연 : 자동 입금 수집이 즉시 반영되지 않을 수 있으므로 일정 시간 후 다시 확인해보시는 것을 권장합니다.

4. 비밀번호 조회

이번 달 예약완료 이용자만 입장 비밀번호를 확인할 수 있습니다.

비밀번호 조회로 이동
  1. 입력 정보 : 이름, 연락처, 동/호수를 신청 정보와 동일하게 입력합니다.
  2. 조회 조건 : 해당 월 예약이 완료되어 있어야 하고, 관리자가 월별 비밀번호를 등록해 두어야 합니다.
  3. 출력 내용 : 해당 월 입장 비밀번호와 입금확인일을 확인할 수 있습니다.

5. 이용 전 꼭 확인해주세요

시스템 사용 시 알아두면 좋은 운영 안내입니다.

  • 공지 확인 : 메인 화면 상단 공지와 전광판에서 운영 안내, 입금자 확인 요청 내용을 수시로 확인해주세요.
  • 입금 순서 : 반드시 예약 신청을 먼저 완료한 뒤 입금해주세요.
  • 자동 입금 확인 시간 : 자동 동기화는 02:00~07:00를 제외한 시간대에 주기적으로 진행됩니다.
  • 개인정보 보관 : 성명, 동/호수, 연락처 정보는 예약 및 입금 확인, 상태 안내를 위해 사용되며 시설 이용 완료 후 12개월 보관됩니다.
  • 도움말 위치 : 이 페이지는 홈페이지 하단 도움말 링크에서 언제든 다시 열 수 있습니다.
